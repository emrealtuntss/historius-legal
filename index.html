Historius Dark History Channel

Automated short-form and long-form video publishing tool for the Historius YouTube channel.

About

Historius is a personal content-publishing tool operated by the creator of the Historius YouTube channel. It automates the workflow of publishing original short-form and long-form documentary content across TikTok, YouTube, Instagram, and Facebook using each platform's official API.

What It Does

  • Renders original dark-history documentary videos produced by the channel operator.
  • Publishes those videos to the operator's own social media accounts.
  • Uses TikTok's Content Posting API to upload videos to the operator's TikTok account.
  • Uses YouTube Data API, Instagram Graph API, and Facebook Graph API for cross-posting.

Who It's For

Historius is a single-operator tool. It is not offered as a public service, does not accept third-party sign-ups, and does not host any content produced by users other than the operator.
